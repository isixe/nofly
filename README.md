# Nofly

一个轻量级、无框架的 Vue 3 和 React 组件示例库，旨在加速前端开发，减少无用组件和依赖。

特性
- 同时支持 Vue 3 和 React
- 轻量级组件集合
- 快速上手，无依赖安装

## 快速开始

### 组件使用

将组件源码复制到你项目的公共 components 目录。例如 `src/components` 或你项目的任意组件目录。

### 引入方式

引入路径根据你的项目配置（例如 `@/components/...`）调整，或者将组件统一放入 `src/components` 并在入口文件做统一导出或注册。
